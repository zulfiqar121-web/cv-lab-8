# Content Based Image Retrieval System

This repository contains multiple Content Based Image Retrieval (CBIR) applications built using Python and OpenCV. The project demonstrates how computer vision techniques can be used to retrieve visually similar images based on features such as color, texture, and patterns.

## Project Overview

The notebook includes different CBIR case studies:

* Texture Based Image Retrieval
* Medical Image Retrieval
* Wildlife Image Matching
* Art Image Similarity Detection
* Industrial Defect Detection

Each task uses feature extraction and similarity matching techniques to identify related images from a dataset.

## Technologies Used

* Python
* OpenCV
* NumPy
* Matplotlib
* Scikit Image
* Scikit Learn

## Features

* Image similarity matching
* Color histogram analysis
* Texture feature extraction
* Local Binary Pattern implementation
* Gabor filter based analysis
* HOG feature extraction
* Cosine similarity comparison

```bash
pip install opencv-python scikit-image scikit-learn matplotlib numpy
```

## How to Run

1. Open the notebook in Google Colab or Jupyter Notebook.
2. Upload the image dataset ZIP file.
3. Run the notebook cells step by step.
4. The system will process the images and retrieve similar results.
